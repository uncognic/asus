# asus
Simple command line tool for fetching and setting system info on ASUS laptops
## Features:
### Get:
- Battery charge limit
- Performance/fan profile
- Fan speed RPM
- CPU temperature celsius
### Set:
- Battery charge limit
- Performance/fan profile
## Support:
Tested on:
- ASUS Vivobook S16 M3607HA

If you have another laptop and it doesn't work, you probably have different /sys/class paths. Contact me and I can add them
